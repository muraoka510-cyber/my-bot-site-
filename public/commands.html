プラン別の対応状況バッジ・例文つき。Free = 無料 Standard = 有料

バッジ
FreeStandard制限あり

「Free + 制限あり」は機能は使えますが、保存期間/回数/間隔などに制約があります。

注意

実行には権限が必要なものがあります（例: Manage Server、WL限定）。Botのロール位置・権限もご確認ください。

困ったら

例文の通りに打ってもうまく動かない時は !debug_perms / !debug_intents を実行して権限やIntentsを確認してください。

自動機能（コマンド不要）
導入直後から常時動作します。必要に応じて各コマンドで調整できます。
アンチニューク（監査ログ監視）
FreeStandard
監査ログで チャンネル削除/作成、ロール改変、Bot追加 などの危険操作を検知し、実行者を即時 BAN/KICK。 WLユーザー/ロールは除外。
招待URL即BAN
FreeStandard
discord.gg/ や discord.com/invite/ を含む投稿を検知すると、メッセージ削除＋投稿者を BAN。
連投（バースト）検知
FreeStandard
1秒ペースの連投（既定は count=10 / window=10s）を検知すると自動処罰。 既定は ロール全剥奪＋ミュート。!burst_* で調整可。
参加直後の厳格監視（Probation）
FreeStandard
参加から一定時間（既定 10分）は URL多投・大量メンション・高速投稿を検知すると Quarantine。 !probation / !log_showcontent などで調整。
CAPTCHA 認証
FreeStandard
参加時にDMでコード送信。未完了は自動で Quarantine。!captcha on|off / !verified_role で調整。
各種ログの自動出力
Free制限ありStandard
参加 / メッセージ削除・編集 / ボイス入退室 / チャンネル・メンバー・サーバー更新 / ピン更新 を自動で記録。 Freeはログ保持 7日、Standardは 30日。!log_set で保存先を指定。
